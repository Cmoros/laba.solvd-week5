## Getting Started

---

### Prerequisites

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Cmoros/laba.solvd-week5.git
   ```
2. Go to the project folder

   ```sh
   cd laba.solvd-week5
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

## Usage

---

To Test All:

```sh
npm t
```

- Test Stack:

  ```
   npm run test:stack
  ```

- Test Stack:
  ```
   npm run test:queue
  ```
